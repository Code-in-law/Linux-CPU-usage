# Linux CPU usage 

Bash-скрипт созданный в рамках обучения по администрированию Linux-систем 

## Функциональность
отображение заггрузки CPU на основе:
<ol>
  <li>us — процент использования CPU пользовательскими процессами;</li>
  <li>sy — процент использования CPU системными процессами;</li>
  <li>ni — процент использования CPU процессами с изменённым приоритетом (nice)</li>
</ol>

## Использование
```bash
chmod +x health.sh
./health.sh
```
