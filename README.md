# Linux CPU usage 

Bash-скрипт созданный в рамках обучения по администрированию Linux-систем 

## Функциональность
- отображение заггрузки CPU на основе:
1) us — процент использования CPU пользовательскими процессами;
2) sy — процент использования CPU системными процессами;
3) ni — процент использования CPU процессами с изменённым приоритетом (nice)

## Использование
```bash
chmod +x health.sh
./health.sh
